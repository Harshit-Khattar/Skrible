# Skrible Project v0.1

## How to Run

1. Navigate to the project directory:
```console
    cd final-project  
    cd v0.1  
    cd skrible
```
2. Activate your Node environment (using conda):
```console
    conda activate node_env  
```
3. Install the dependencies:
```console
    npm install  
```
4. Run the development server:
```console
    npm run dev  
```
